**Geospatial Data Visualization App**

This is a Streamlit application that allows you to visualize randomly generated geospatial data on an interactive map.

**Features**:
Data Generation: Generates random latitude and longitude data points.

Customizable Data Ranges: Control the range of generated latitudes and longitudes.

Number of Data Points: Specify the number of random data points.

Interactive Map: Displays data points on a Folium map.

Styled Markers: Data points are represented by styled markers.

Tooltips: Hover over markers to see coordinates.

Zoom Control: Adjust the initial zoom level of the map.

Data Preview: Displays a sample of the generated data.
